What is pyStocks?

pyStocks is a stock portfolio site that you can use to practice your skills in the market- with realtime data.

It has been developed in python utilizing Flask.

To gather realtime market data, pyStocks uses Finnhub.io , an amazing free API.

This project is hosted using digital ocean, on the same server as my personal website.

If you have any questions or concerns, head over to my site and shoot me a message!
